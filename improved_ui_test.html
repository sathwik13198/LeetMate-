Two Sum
This solution uses a hash map to keep track of numbers we've seen so far. For each number, we check if its complement (target - num) is already in the hash map. If it is, we found our pair! If not, we add the current number to the hash map and continue.
Solution 
def twoSum(nums, target):
    # Create a dictionary to store numbers we've seen
    seen = {}
    
    # Iterate through the array with index
    for i, num in enumerate(nums):
        # Calculate the complement we need to find
        complement = target - num
        
        # Check if the complement exists in our dictionary
        if complement in seen:
            return [seen[complement], i]
        
        # Store current number and its index
        seen[num] = i
    
    # No solution found
    return []
Time Complexity: O(n) | Space Complexity: O(n)
