869. Reordered Power of 2
This solution checks if the digits of the given number can be reordered to form a power of 2. We use a counter to track the frequency of each digit in the input number, then compare it with the digit frequencies of all powers of 2 within the valid range.
Solution 
from collections import Counter

class Solution:
    def reorderedPowerOf2(self, n: int) -> bool:
        # Count the frequency of each digit in n
        n_counter = Counter(str(n))
        
        # Check all powers of 2 with the same number of digits
        num_digits = len(str(n))
        
        # Start from 2^0 and go up to a reasonable limit
        power = 0
        while True:
            # Calculate the current power of 2
            current_power = 1 << power
            
            # If we've exceeded the digit count, break
            if len(str(current_power)) > num_digits:
                break
            
            # If we have the same number of digits, check if the digit counts match
            if len(str(current_power)) == num_digits:
                if Counter(str(current_power)) == n_counter:
                    return True
            
            power += 1
        
        return False
